# 🏙️ Smart City Weather Dashboard

## 📌 Description
The **Smart City Weather Dashboard** is a high-aesthetic, professional, and data-rich application. Transformed with a premium dark-themed glassmorphism design, this dashboard provides a cinematic, multi-directional animated intro sequence that seamlessly transitions into a comprehensive weather data center. 

Unlike basic weather apps, this application offers an immersive experience featuring advanced data visualizations, interactive environmental analytics, and real-time monitoring of city-wide metrics.

---

## 🚀 Core Features

- **🎬 Cinematic Intro Sequence:** A premium, multi-directional animated intro sequence prior to viewing dashboard telemetry.
- **✨ Glassmorphism UI:** Sophisticated dark-theme aesthetic featuring translucent cards, modern typography, and smooth state transitions.
- **🗺️ Air Quality Zone Map:** Interactive visualization of localized air quality and environmental health metrics.
- **📡 Sensor Network Panel:** Real-time status indicators representing advanced city IoT sensor networks.
- **📊 Advanced Analytics:** Comprehensive weather data reporting (temperature, humidity, pressure, visibility, etc.).
- **📍 Geolocation:** Automatic zero-touch localization using the browser Geolocation API.
- **👕 Smart Lifestyle Advice:** Context-aware suggestions (e.g., clothing, activities) based on intelligent analysis of current conditions.
- **📱 Responsive Layout:** Precision-engineered to deliver a seamless experience across desktop, tablet, and mobile.

---

## 🌐 Public APIs Used

**OpenWeatherMap API**  
https://openweathermap.org/api

Endpoints in use:
- Current Weather API
- 5 Day / 3 Hour Forecast API

---

## 🛠️ Tech Stack

- **Framework:** React.js (v18)
- **Tooling:** Vite for lightning-fast HMR and optimized builds
- **Styling:** CSS3 Variables, Glassmorphism Techniques, Advanced Keyframe Animations
- **Data Source:** OpenWeatherMap REST API
- **Location Services:** HTML5 Geolocation API

## ⚙️ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/smart-city-weather-dashboard.git
   cd smart-city-weather-dashboard
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

*(Note: Ensure you configure your OpenWeatherMap API key in the application's environment configuration before running).*
